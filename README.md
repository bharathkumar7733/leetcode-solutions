# LeetCode Practice Solutions 🚀

Each directory is named after the LeetCode problem ID and title, containing the solution file along with any notes or complexities.

- `121-best-time-to-buy-and-sell-stock/`
- `283-move-zeroes/`
- `412-fizz-buzz/`
